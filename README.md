# A portable light printed using Rapid Liquid Printing

## Specs

- 2700K
- About 400 lumens
- Dimmable
- Rechargeable
- Flashes when battery low

## Implementation

- Micro: RP2040
- LED Driver: TPS61165
- LED: OSRAM GW JTLPS1.EM-KKLL-XX58-1
